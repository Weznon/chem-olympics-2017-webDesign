  • Formation
    • Methenolone is a derivative of dihydrotestosterone (DHT). It differs in the presence of a double bond at carbon one and two, which in turn greatly increases the hormone’s anabolic nature. It also carries an added 1-methyl group that protects it from being metabolized by the liver, resulting in easier detection
  • Structure
    • C22H32O3

  • Effects
    • Methenolone’s primary effect is that it enables the user to retain nitrogen by inhibiting the liver’s functions of converting nitrogen into molecules which are passed in urine. Methenolone is also known to help build lean muscle via binding to an androgen receptor. This two effects result in the user being able to build muscle more efficiently.
  • Detection
    • Since Methenolone is not metabolized by the liver, it can be detected using liquid chromatography or gas spectroscopy.
